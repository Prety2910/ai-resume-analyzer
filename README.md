</p>

---

##  Overview

AI Resume Analyzer helps candidates evaluate their resumes through ATS-style scoring, job-role matching, missing skill detection, and improvement recommendations.

It also includes an interactive chatbot that answers questions about your resume and suggests ways to improve it.

---

##  Features

✅ Upload Resume in PDF Format  
✅ ATS Resume Score  
✅ Best Job Role Prediction  
✅ All Role Match Scores  
✅ Detected Skills Extraction  
✅ Missing Skills Detection  
✅ Resume Improvement Tips  
✅ Resume Chat Assistant  
✅ Clean Premium Dashboard UI
